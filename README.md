# FlavorFindAI

Live Website: https://flavorfindai.vercel.app/

## About

FlavorFindAI is an AI-powered recipe recommendation platform designed to help users discover recipes based on ingredients and cuisine preferences. The platform analyzes food datasets and uses machine learning techniques to suggest suitable recipes for users.

The goal of FlavorFindAI is to simplify cooking decisions by helping users find dishes they can prepare using the ingredients they already have. It provides an interactive and easy-to-use interface where users can explore different recipes and get intelligent suggestions.

---

## Features

- AI-powered recipe recommendation system
- Ingredient-based recipe suggestions
- Clean and responsive user interface
- Multiple recipe result pages
- Fast cloud deployment

## Installation

Clone the repository

```bash
git clone https://github.com/Pravinsaichiday/RecipeRecommendation_Project.git
```

Go to the project folder

```bash
cd RecipeRecommendation_Project
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
